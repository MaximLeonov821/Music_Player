# Future Past

# Описание проекта
В данном проекте я и моя команда создали музыкальный плеер в стиле ретрофутуризма. Чтобы добавить музыку в этот плеер, ее нужно скачать на свой телефон, и далее она автоматически добавится на главную страницу со всей скачанной музыкой. Музыку можно загружать на любой вкус и цвет, и она будет работать даже без интернета и всяких подписок(приложение абсолютно бесплатное). Также в вариации есть 6 разных тем - если не понравилась одна - у вас в запасе еще 5 различных тем приложения. Музыку можно добавлять отдельно в избранное, чтобы наслаждаться только тем, что вам в данный момент по душе. Также, если включить интернет на своем телефоне, можно будет автоматически подгружать текст песне, которая играет на данный момент (увы, но для русскоязычных треков текст не находится). Также в определенных темам обложка будет анимированная, то есть при прослушивании и паузе обложка будет реагировать на ваши действия с треком. Еще реализована функция перемешивания треков - каждый раз играет случайная песня.  В любой момент трек можно удалить из плеера, и соответсвенно он удалится из телефона - экономия вашего пространства. 

# Использованные технологии:
1) Язык написания программы: Kotlin;
2) Среда разработки: Android Studio;
3) Использолвание Genius API для онлайн-вытягивания текста для песен;
4) Библиотеки: serialization для текста, lottie для анимаций обложек треков
